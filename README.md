## Description

A Simple Website Page to learn Bubbling, DOM Traversing, Lazy Loading Images and more. The CSS and HTML portion of this was imported from a lecture, and only the JavaScript was edited by me.

## Usage Information

All of the functions on this project are easily re-usable in future websites and well optimised to reduce CPU usage. This website includes: <br>

1. Smooth scrolling through JavaScript rather than CSS <br>
2. Button functionality <br>
3. Sticky NavBar <br>
4. Highlight elements on the NavBar upon mouseover <br>
5. Lazy Loading Images for slower internet connections <br>
6. Tabbed elements - as seen under the Operations section <br>
7. Slider elements - as seen under the Testimonials section <br>
